# Data

This repository does not redistribute ToyADMOS audio files.

Download ToyADMOS from the official source and place the `ToyConveyor` IND clips in this layout:

```text
data/ToyConveyor/
  case1/
    NormalSound_IND/
    AnomalousSound_IND/
  case2/
    NormalSound_IND/
    AnomalousSound_IND/
  case3/
    NormalSound_IND/
    AnomalousSound_IND/
```

The experiments in this repository focus on binary `normal` vs `anomalous` detection for `ToyConveyor` `IND` clips.
